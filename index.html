About PyHack

PyHack is a dynamic, community-driven workshop designed to teach game development using Python.

FAQs

PyHack is a comprehensive Python game development workshop designed to teach participants how to create engaging games using Python. It covers everything from basic game mechanics to advanced programming concepts.

Our workshops are conducted both online and in-person at various locations. Online sessions are conducted via interactive video platforms, while in-person workshops are held at tech hubs and educational institutions across major cities.

You'll learn game mechanics, physics simulation, AI algorithms, 2D/3D graphics, animation techniques, sound integration, user interface design, and how to deploy games across multiple platforms.

Basic Python knowledge is recommended but not required. We provide beginner-friendly modules and our instructors will guide you through the fundamentals needed for game development.
